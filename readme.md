# Github Helper Bot

一个基于ncatbot的qq群 Github 辅助机器人

该机器人会在指定的 Github 仓库中监听事件：

- 当有新的 Issue 被创建时，会将其发送到指定的 qq 群中。
- 当有新的 Pull Request 被创建时，会将其发送到指定的 qq 群中。
- 当有新的 Issue 被接受时，会将其发送到指定的 qq 群中。
- 当有新的 Bug Issue 被确认时，会将其发送到指定的 qq 群中。
- 当有新的 Enhancement Issue 被接受时，会将其发送到指定的 qq 群中。
- 当对话中包含`#XXX`时，会将其作为 Issue 编号，查询该 Issue 的详情，并将其发送到指定的 qq 群中。
